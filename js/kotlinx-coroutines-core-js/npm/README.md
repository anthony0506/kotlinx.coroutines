# kotlinx.coroutines 

Library support for Kotlin coroutines in
[Kotlin/JS](https://kotlinlang.org/docs/reference/js-overview.html).

```kotlin
launch {
    delay(1000)
    println("Hello from Kotlin Coroutines!")
}
```
